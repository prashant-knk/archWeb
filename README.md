# Kala & Kriti Website

A beautiful, minimalist website for Kala & Kriti (K&K) - a premier architectural model-making company. The website features a brown and beige color scheme matching the original PDF presentation.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Brown & Beige Color Scheme**: Elegant minimalist design matching the PDF presentation
- **Smooth Animations**: Fade-in animations and smooth scrolling
- **Image Gallery**: Showcase of architectural models and projects
- **Modern UI/UX**: Clean, professional design with excellent user experience

## File Structure

```
webKnK/
├── index.html          # Main HTML file
├── styles.css          # CSS styles with brown/beige color scheme
├── script.js           # JavaScript for interactivity
├── images/             # Extracted images from PDF
├── pdf_metadata.json   # Metadata from PDF extraction
└── README.md           # This file
```

## Getting Started

1. Simply open `index.html` in your web browser
2. No build process or dependencies required
3. All images are referenced from the `images/` directory
